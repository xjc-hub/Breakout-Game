2.3 碰撞检测算法
圆形与矩形碰撞检测：
─────────────────────────────────────────────────────

步骤1：找到矩形上距离圆心最近的点

        圆心 ●
             │
        ┌────┼────┐
        │    ↓    │
        │  最近点  │
        │    ●    │
        └─────────┘

算法：
closestX = clamp(ball.x, rect.x, rect.x + rect.width)
closestY = clamp(ball.y, rect.y, rect.y + rect.height)

步骤2：计算距离

distance = sqrt((ball.x - closestX)² + (ball.y - closestY)²)

步骤3：判断碰撞

if distance < ball.radius:
    发生碰撞！

步骤4：确定碰撞方向

    ┌─────────────────────────────────────┐
    │                                     │
    │         distTop (上边距)            │
    │              ↓                      │
    │    ┌─────────●─────────┐            │
    │    │         │         │ ← distLeft │
    │    │    球心 ●         │   distRight│
    │    │                   │            │
    │    └─────────●─────────┘            │
    │              ↑                      │
    │        distBottom (下边距)          │
    │                                     │
    └─────────────────────────────────────┘

比较 min(distLeft, distRight) 与 min(distTop, distBottom)
较小的方向即为碰撞方向

7. 数学基础
7.1 向量运算
向量加法（位移）：
position = position + speed × deltaTime

向量长度（速度大小）：
|speed| = √(speed.x² + speed.y²)

向量归一化（方向）：
normalize(speed) = speed / |speed|
7.2 碰撞检测数学
点到矩形最近点：
closestX = clamp(point.x, rect.left, rect.right)
closestY = clamp(point.y, rect.top, rect.bottom)

两点距离：
distance = √((x1-x2)² + (y1-y2)²)

圆与矩形碰撞条件：
distance < circle.radius
7.3 三角函数应用
角度转弧度：
radians = degrees × π / 180

极坐标转直角坐标：
x = magnitude × cos(angle)
y = magnitude × sin(angle)

用于反弹角度计算：
speed.x = speed × cos(bounceAngle)
speed.y = speed × sin(bounceAngle)
