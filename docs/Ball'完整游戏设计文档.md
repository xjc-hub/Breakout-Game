3. 物理引擎设计
3.1 重力系统
物理原理：
─────────────────────────────────────────────────────────────────

牛顿第二定律：F = ma（力 = 质量 × 加速度）

在游戏中，我们简化为：
    速度变化：v = v + a × dt
    位置变化：x = x + v × dt

其中：
    a = gravity（重力加速度）
    dt = 1（一帧的时间间隔，简化处理）

void Ball::ApplyGravity() {
    // 每帧增加向下的速度
    speed.y += gravity;
    
    // 限制最大速度，防止速度过快
    float currentSpeed = sqrt(speed.x² + speed.y²);
    if (currentSpeed > maxSpeed) {
        // 归一化并缩放
        speed.x = (speed.x / currentSpeed) * maxSpeed;
        speed.y = (speed.y / currentSpeed) * maxSpeed;
    }
}

void Ball::BouncePaddle(Rectangle paddleRect) {
    // 只有下落时才检测碰撞
    if (speed.y <= 0) return;
    
    // 检测是否在球拍范围内
    if (碰撞检测通过) {
        
        // 计算撞击点 (-1 到 +1)
        float hitPoint = (ball.x - paddle_center_x) / (paddle_width / 2);
        hitPoint = clamp(hitPoint, -1.0f, 1.0f);  // 限制范围
        
        // 计算反弹角度
        float angle = 90.0f - hitPoint * 50.0f;  // 40° ~ 140°
        float angleRad = angle * PI / 180.0f;
        
        // 计算新速度
        float speedMagnitude = sqrt(speed.x² + speed.y²) + bounceForce;
        speed.x = speedMagnitude * cos(angleRad);
        speed.y = -speedMagnitude * sin(angleRad);  // 负号表示向上
        
        // 修正位置，防止穿透
        position.y = paddleRect.y - radius;
    }
}

4.2 碰撞检测代码

bool Ball::CheckBrickCollision(Rectangle brickRect) {
    // 步骤1：找到最近点
    float closestX = std::max(brickRect.x, 
                    std::min(position.x, brickRect.x + brickRect.width));
    float closestY = std::max(brickRect.y, 
                    std::min(position.y, brickRect.y + brickRect.height));
    
    // 步骤2：计算距离
    float distX = position.x - closestX;
    float distY = position.y - closestY;
    float distance = std::sqrt(distX * distX + distY * distY);
    
    // 步骤3：判断碰撞
    if (distance < radius) {
        // 步骤4：确定碰撞方向
        float distLeft   = position.x - brickRect.x;
        float distRight  = brickRect.x + brickRect.width - position.x;
        float distTop    = position.y - brickRect.y;
        float distBottom = brickRect.y + brickRect.height - position.y;
        
        float minDistX = std::min(distLeft, distRight);
        float minDistY = std::min(distTop, distBottom);
        
        if (minDistX < minDistY) {
            // 水平碰撞
            speed.x *= -1;
        } else {
            // 垂直碰撞
            speed.y *= -1;
        }
        
        return true;
    }
    return false;
}

6.2 加速原理
加速移动实现：
─────────────────────────────────────────────────────────────────

基础原理：
    移动距离 = 速度 × 时间
    
    每帧移动：position.x += speed
    （假设每帧时间固定为 1/60 秒）

加速实现：
─────────────────────────────────────────────────────────────────

    // 定义速度常量
    float normalSpeed = 18.0f;   // 普通速度：每帧移动 18 像素
    float boostSpeed  = 28.0f;   // 加速速度：每帧移动 28 像素

    // 每帧检测输入
    float currentSpeed = normalSpeed;  // 默认普通速度

    // 检测是否按住 Shift
    if (IsKeyDown(KEY_LEFT_SHIFT) || IsKeyDown(KEY_RIGHT_SHIFT)) {
        currentSpeed = boostSpeed;  // 切换到加速
    }

    // 根据方向键移动
    if (IsKeyDown(KEY_LEFT)) {
        paddle.x -= currentSpeed;  // 向左移动
    }
    if (IsKeyDown(KEY_RIGHT)) {
        paddle.x += currentSpeed;  // 向右移动
    }

9.2 帧率与时间
帧率（FPS）是什么？
─────────────────────────────────────────────────────────────────

FPS = Frames Per Second（每秒帧数）

60 FPS 表示：
    - 每秒绘制 60 帧
    - 每帧间隔：1000ms / 60 ≈ 16.67ms
    - 游戏循环每秒执行 60 次

SetTargetFPS(60) 的作用：
─────────────────────────────────────────────────────────────────

    告诉 Raylib 在每帧结束后等待，确保帧率稳定

─────────────────────────────────────────────────────────────────

    • 游戏类型：动作游戏通常 60 FPS

11. 时间衰减计分系统
11.1 设计原理
为什么需要时间衰减计分？
─────────────────────────────────────────────────────────────────

传统计分问题：
    - 每个砖块固定分数，无论何时击破都一样
    - 玩家没有紧迫感，可以慢慢玩
    - 技术好的玩家和技术差的玩家分数差距不明显

时间衰减计分优势：
    - 鼓励玩家快速通关
    - 技术好的玩家获得更高分数
    - 增加游戏挑战性和重玩价值

/**
 * 计算时间衰减分数
 * 
 * @param baseScore  砖块基础分
 * @param gameTime   游戏经过时间（秒）
 * @return           衰减后的最终分数
 */
int CalculateScore(int baseScore, float gameTime) {
    // 计算衰减倍率
    float multiplier = 5.0f - gameTime * 0.05f;
    
    // 限制最低倍率为 1.0（分数不会低于基础分）
    if (multiplier < 1.0f) multiplier = 1.0f;
    
    // 计算最终分数（取整）
    return (int)(baseScore * multiplier);
}

// 使用示例
int earnedScore = CalculateScore(10, gameTime);  // gameTime 是游戏经过的秒数
score += earnedScore;

12.1 设计原理
为什么需要落地惩罚？
─────────────────────────────────────────────────────────────────

问题：
    - 原版游戏小球落地只扣生命
    - 分数不受影响，玩家可能"破罐破摔"

    - 增加游戏紧张感

13.4 文件读写实现

/**
 * 从文件加载排行榜
 * 
 * 文件格式：每行一个条目
 * 格式：名称 分数 时间戳
 * 示例：Player 450 1709234567
 */
void Load() {
    FILE* f = fopen(filename, "r");
    if (f) {
        count = 0;
        // 逐行读取，直到文件结束或达到最大数量
        while (count < MAX_ENTRIES && 
               fscanf(f, "%31s %d %ld", 
                      entries[count].name, 
                      &entries[count].score, 
                      &entries[count].timestamp) == 3) {
            count++;
        }
        fclose(f);
    }
}

/**
 * 保存排行榜到文件
 */
void Save() {
    FILE* f = fopen(filename, "w");
    if (f) {
        for (int i = 0; i < count; i++) {
            fprintf(f, "%s %d %ld\n", 
                    entries[i].name, 
                    entries[i].score, 
                    entries[i].timestamp);
        }
        fclose(f);
    }
}

/**
 * 发射小球
 * 
 * 随机机制：
 * 1. 角度：60° ~ 120° 随机选择
 * 2. 方向：50% 向左，50% 向右
 * 3. 速度：基础速度 + 轻微随机变化
 */
void Ball::Launch(float paddleX, float paddleWidth) {
    if (launched) return;
    
    // 随机角度：60° ~ 120°
    float angle = (rand() % 61 + 60) * 3.14159f / 180.0f;
    
    // 随机决定向左还是向右
    if (rand() % 2 == 0) {
        angle = 3.14159f - angle;  // 镜像角度
    }
    
    // 基础速度 + 随机变化 (-1.0 ~ +1.0)
    float baseSpeed = 8.0f;
    float speedVariation = (rand() % 5 - 2) * 0.5f;
    float launchSpeed = baseSpeed + speedVariation;
    
    // 计算速度分量
    speed.x = launchSpeed * cos(angle);
    speed.y = -launchSpeed * abs(sin(angle));  // 负号表示向上
    
    launched = true;
}
